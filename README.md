# 블로그 썸네일 메이커

사진에 텍스트를 올려 블로그 썸네일을 무료로 만드는 웹 앱입니다.

**[thumbnail.heeso.site](https://thumbnail.heeso.site)**

## 기능

- 이미지 업로드 후 텍스트 오버레이
- 폰트 선택 (지마켓Sans, 꾸불림, 오늘원콘콘, 이서윤)
- 폰트 크기 조절 (±5단계)
- 텍스트 색상 (흰색 / 검정색)
- 텍스트 테두리 ON/OFF
- 이미지 어둡기 조절
- **1:1 미리보기** — 블로그에 정사각형으로 표시될 때 어떻게 보이는지 확인
- JPG로 다운로드

## 기술 스택

Vanilla JS / HTML / CSS — 빌드 도구 없이 정적 파일만으로 동작합니다.

## 로컬 실행

별도 설치 없이 `index.html`을 브라우저에서 열면 됩니다.

```bash
open index.html
```

또는 간단한 로컬 서버를 띄워도 됩니다.

```bash
npx serve .
```
